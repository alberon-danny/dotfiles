Various useful scripts and tools, designed to be installed via composer.

Example installation configuration
```json
{
    "name" : "dannycain/utils",
    "require" : {
        "dannycain/scripts": "dev-master"
    },
    "repositories" : [
        {
            "type" : "git",
            "url" : "git@gitlab.com:danny-tools/scripts.git"
        }
    ],
    "autoload" : {

    }
}
```
